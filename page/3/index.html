Java入门

02 初识$Java$

  • 一门高级编程语言
  • 由$sun（Stanford\ University\ Network）$研发，现属于$Oracle$
  • 詹姆斯·高斯林$(James\ Gosling)$被称为$Java$之父
  • 主要做互联网系统开发
  • 分为$SE\ EE\ ME$

03 $JDK$的选择安装与下载

  • 使用$Java$需要$JDK(Java\ Development\ Kit)$开发者工具包

  • $LTS$版本有8,11,17

  • 在$cmd$中使用

    • $java$
    • $java -version$
    • $javac -version$

    检查安装是否成功,其中前者为执行工具,后者为编译工具

04 掌握$DOS$窗口常见命令的使用

  • $C:$切盘符
  • $dir$查看当前路径下的文件信息
  • $cd$
    • 跟随当前盘符下的路径:$cd\ $xxx
    • 回退:$cd\ ..$
    • 回退到根目录:$cd\ $\
  • $cls$清屏

05 开发$HelloWorld$程序

  • $code:$
  • 编译运行,代码目录下打开$cmd$,输入
    • $javac\ 文件名$
    • $java\ 文件名去后缀$
  • $else$
    • $code$文件扩展名为$java$
    • 文件名与代码类名称一致
    • 使用英文

06 常见程序问题解答

  • 没改文件类型(扩展名)
  • 没$save$
  • 文件名与类名不一致
  • 大小写错误,符号错误,没有$main$方法
  • 括号不配对
  • 编译执行工具使用不当

07 $Java$程序的执行原理,$BUG$介绍

  • 机器语言<–汇编语言<–高级语言
  • 格蕾丝·赫伯$(Grace\ Murray\ Hopper)$

08 $JDK$组成,跨平台原理

  • $JVM(Java\ Virtual\ Machine)$+核心类库–>$JRE(Java\ Runtime\ Environment)$
  • $java,javac$+$JRE$–>$JDK(Java\ Development\ Kit)$
  • $Class$文件跨平台只需要不同系统下的$JVM$

09 $Path$,$JAVA_HOME$环境变量的配置

  • $Path$环境变量用于命令行在任意目录启动特定目录下的程序(路径常为$bin$文件夹)
  • 注意检查是否配置成功
  • $JAVA_HOME$用于指出$JDK$安装位置
  • 在配置可以使用$%$xxx$%$来引用已有的路径,如将$Path$配置为$Path\ %JAVA_HOME%$\$bin$

10 $IntelliJ\ IDEA$开发工具概述

  • 一款开发工具
  • 傻瓜安装即可

11 $IDEA$开发工具概述

  • $project$–>$module$–>$package$–>$class$
  • $package$包一般以倒序域名命名,如$com.ithema.$xxx
  • 创建$class$时不必写后缀,编译后的$class$文件位于$out$文件夹里

12 $IDEA$的配置,快捷键设置

  • 快捷键
    • $main,psvm,sout$用于快速键入代码
    • $Ctrl\ +\ D$ 复制当前行到下一行
    • $Ctrl\ +\ Y$ 删除当前行
    • $Ctrl\ +\ ALT\ +\ L$ 格式化代码
    • $ALT\ +\ SHIFT\ + \uparrow/\downarrow$ 上下移动当前代码
    • $Ctrl\ +\ /$ 注释当前行, $Ctrl\ +\ SHIFT\ +\ /$ 插入多行注释

13 $IDEA$的其他操作

  • 修改可能是$refactor$,再$rename$
  • 导入一般新建$module$到同一$project$后直接复制$codes$
  • 在$IDEA$里$remove$后检查是否真的删除

阶段总结:Java概述,快速入门,IDEA使用

14 基础语法-注释详解

  • 注释用于解释说明代码

  • 分类

    • 单行注释:$//$
    • 多行注释:$/\ /$
    • 文档注释:$/*\ /$

  • 注释特点:不影响程序运行,编译后的$class$文件中没有注释

  • 快捷键:

    • $Ctrl\ +\ /$ 当前行注释
    • $Ctrl\ +\ Shift\ + /$ 对选中代码多行注释

15 基础语法-字面量详解

  • 字面量是数据在程序中的书写格式,分为
    • 整数
    • 小数
    • 字符, 用单引号
    • 字符串, 用双引号
    • 布尔值, $true\ flase$
    • 空值, $null$
  • 常见特殊值
    • $true\ false\ null\ \textbackslash n \ \textbackslash t$

16 基础语法-变量详解

  • 用于储存数据, 本质是内存中的一块区域
  • 完整定义为: 数据类型 变量名称 = 数据;

17 基础语法-变量使用时的注意事项

  • 先声明后使用
  • 变量和数据的类型统一
  • 不可重名
  • 注意使用的变量需要初始化

18 基础语法-关键字,标识符

  • 关键字为$Java$内定的词, 不可用作类名, 变量名
  • 标识符即为名字, 不可以数字开头, 不可使用关键字

Java数组

02 静态初始化数组，数组的访问

  • 写法
    • 1.数据类型[] 数组名={元素1，元素2，…}；
    • 2.数据类型[] 数组名=new 数据类型[]{元素1，元素2，…}；
  • 数据类型[] 数组名 也可写成 数据类型 数组名[]
  • 数组属于引用数据类型，储存数组在内存中的地址
  • 访问
    • 元素：数组名[索引]
    • 长度：数组名.length
    • 最大索引：长度-1（长度>0）
    • 访问超出最大索引：索引越界异常

    • 整型 0
    • 浮点型 0.0
    • 布尔型 false

  • 静态与动态
    • 静态适合元素量已知业务
    • 动态适合元素量未知业务

05 数组的内存执行原理

  • JVM分区
    • 方法区
    • 栈内存
    • 堆内存
  • int a与int[] arr=new int[3]的区别
    • a是变量，放在栈里，存储为a的值
    • new int[3]创建数组对象，在堆内存开辟3个int
    • arr是变量，在栈中，存储为堆内存的地址
  • 多个数组变量只向同一数组对象
    • 数组变量储存的是同一地址
    • 用任意数组变量修改数组内元素，其他均受影响
  • 存null表示没有指向对象，此时不能访问

Java语法

02 变量里的数据在计算机中的储存原理

  • 数据在计算机底层采用二进制储存
  • 除二取余法可以快速计算出二进制数
  • 最小数据单位为字节，1B=8b

03 字符，图片，声音的储存说明

  • 字符使用ASCII码储存
  • 图片和声音也使用二进制编码

04 二进制，八进制，十六进制

  • 二进制到十进制：每位加权再相加
  • 二进制到八进制：每三位作为一个单元
  • 二进制到八进制：每四位作为一个单元
  • 1TB=1024GB,1GB=1024MB,1MB=1024KB,1KB=1024B,1B=8b

05 数据类型

  • 基本数据类型
    • 整型：byte short int long
    • 浮点型：float double
    • 字符型：char
    • 布尔型：boolean
  • 引用数据类型：String
  • 整数默认为int，小数默认为double

06 自动类型转换

  • 类型转换实现不同变量的赋值
  • 类型范围小的变量可以直接赋值给类型范围大的变量

07 表达式的自动类型转换

  • 小范围的类型会自动转换成大范围的类型运算
  • 最终类型由表达式中最高类型决定
  • byte short char都是转换成int再运算

08 强制类型转换

  • 默认情况下，大范围类型的变量直接赋值给小范围类型的变量会报错

  • 可以强行将类型范围大的变量和数据赋值给类型范围小的变量

  • 强制类型转换可能出现数据丢失

  • 小数强制转换成整数是直接截断小数保留整数

09 算术运算符，+符号做连接运算符

  • 算术运算符

  • 两个整数使用/结果一定是整数

  • +可以作为字符串连接符，规则为能算则算，不能算就在一起

10 自增，自减运算符

  • ++/–对当前变量值+1/-1
  • 前置，先运算后使用；后置，先使用后运算

流程控制

02 分支结构：if，switch，switch穿透性

  • if可以根据条件选择执行某段程序
  • 可以写作
    • if
    • if else
    • if else
      • if else…
  • if泛用性强于swtich，switch在范围内性能较好
  • switch注意事项
    • 支持byte,short,int,char,枚举,String；不支持double,float,long
    • case只能是字面量不能是变量，不能重复
    • 正常使用要写break
  • 穿透性可用于多种入口的情形，简化代码

03 循环结构：for循环，for循环案例

  • for（初始化；循环条件；迭代语句）{循环内容}
  • 应用
    • 重复劳动
    • 灵活执行

04 循环结构：while循环，珠穆朗玛峰案例

  • while（条件）{循环；迭代}
  • while与for
    • 功能一致
    • 具体次数用for，不确定用while

05 循环结构：do-while循环

  • do-while是执行后判断，可以保证至少执行一次

06 死循环，循环嵌套

  • 死循环可用于服务器程序
  • 嵌套解决复杂问题

07 跳转关键字：break，continue

  • break结束当前循环，结束swtich分支
  • continue前往下次循环
