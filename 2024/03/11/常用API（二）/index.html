    • 常见方法

  • Runtime

    • 代表程序所在的运行环境

    • Runtime是一个单例类

    • 常见方法

  • BigDecimal

    • 用于解决浮点数失真问题

    • 常见构造器

    • 常用方法

03 传统时间：Date日期类、SimpleDateFormat

  • Date

    • 代表日期和时间

    • 构造器

    • 常用方法

  • SimpleDateFormat

    • 代表简单日期格式化，可以用来把日期对象、时间毫秒值格式化成我们想要的形式

    • 构造器

    • 常用方法

    • 常见符号

      • y-年
      • M月
      • d-日
      • H-时
      • m-分
      • s-秒
      • EEE-星期
      • a-上午/下午

04 传统时间：秒杀案例、Calendar

  • 秒杀活动

  • Calendar

    • 代表的时系统此刻时间对应的日历，可以单独获取、修改年月日 时分秒

    • 常用方法

05 JDK8开始新增的日期、时间

  • 传统时间类的问题
    • 设计不合理，使用不方便
    • 都是可变对象，修改后会丢失最开始的时间信息
    • 线程不安全
    • 不能精确到纳秒，只能精确到毫秒
  • 代替Calendar
    • LocalDate：年月日
    • LocalTime：时分秒
    • LocalDateTime：年月日时分秒
    • ZoneId：时区
    • ZoneDateTime：带时区的时间
  • 代替Date
    • Instant：时间戳
  • 代替SimpleDateFormat
    • DateTimeFormatter：用于时间的格式化和解析
  • 其他补充
    • Period：时间间隔（年月日）
    • Duration：时间间隔（时分秒纳秒）

06 LocalDate、LocalTime、LocalDateTime、ZoneId、ZoneDateTime、Instant

  • 通用的获取对象的方法

  • LocalDate

    • 处理年月日、星期

    • 常用方法

  • LocalTime

    • 处理时分秒、纳秒

    • 常用方法

  • LocalDateTime

    • 处理年月日、时分秒、星期、纳秒
    • 常用方法

  • ZoneId

    • 常用方法

  • ZoneDateTime

    • 常用方法

  • Instant

    • 通过获取Instant的对象可以拿到此刻的时间，为1970.1.1 0:00到此刻的秒数+纳秒数

    • 可以记录代码执行时间、记录用户操作时间点

    • 常用方法

07 DateTimeFormatter、Period、Duration

  • DateTimeFormatter

    • 常用方法

  • Period

    • 可以用于计算两个LocalDate对象相差的年月日

    • 构造器

  • Duration

    • 可以用于计算两个时间对象相差的天数、小时数、分数、秒数、纳秒数

    • 支持LocalTime、LocalDateTime、Instant

    • 常用方法
