Java入门

02 初识$Java$

  • 一门高级编程语言
  • 由$sun（Stanford\ University\ Network）$研发，现属于$Oracle$
  • 詹姆斯·高斯林$(James\ Gosling)$被称为$Java$之父
  • 主要做互联网系统开发
  • 分为$SE\ EE\ ME$

03 $JDK$的选择安装与下载

  • 使用$Java$需要$JDK(Java\ Development\ Kit)$开发者工具包

  • $LTS$版本有8,11,17

  • 在$cmd$中使用

    • $java$
    • $java -version$
    • $javac -version$

    检查安装是否成功,其中前者为执行工具,后者为编译工具

04 掌握$DOS$窗口常见命令的使用

  • $C:$切盘符
  • $dir$查看当前路径下的文件信息
  • $cd$
    • 跟随当前盘符下的路径:$cd\ $xxx
    • 回退:$cd\ ..$
    • 回退到根目录:$cd\ $\
  • $cls$清屏

05 开发$HelloWorld$程序

  • $code:$
  • 编译运行,代码目录下打开$cmd$,输入
    • $javac\ 文件名$
    • $java\ 文件名去后缀$
  • $else$
    • $code$文件扩展名为$java$
    • 文件名与代码类名称一致
    • 使用英文

06 常见程序问题解答

  • 没改文件类型(扩展名)
  • 没$save$
  • 文件名与类名不一致
  • 大小写错误,符号错误,没有$main$方法
  • 括号不配对
  • 编译执行工具使用不当

07 $Java$程序的执行原理,$BUG$介绍

  • 机器语言<–汇编语言<–高级语言
  • 格蕾丝·赫伯$(Grace\ Murray\ Hopper)$

08 $JDK$组成,跨平台原理

  • $JVM(Java\ Virtual\ Machine)$+核心类库–>$JRE(Java\ Runtime\ Environment)$
  • $java,javac$+$JRE$–>$JDK(Java\ Development\ Kit)$
  • $Class$文件跨平台只需要不同系统下的$JVM$

09 $Path$,$JAVA_HOME$环境变量的配置

  • $Path$环境变量用于命令行在任意目录启动特定目录下的程序(路径常为$bin$文件夹)
  • 注意检查是否配置成功
  • $JAVA_HOME$用于指出$JDK$安装位置
  • 在配置可以使用$%$xxx$%$来引用已有的路径,如将$Path$配置为$Path\ %JAVA_HOME%$\$bin$

10 $IntelliJ\ IDEA$开发工具概述

  • 一款开发工具
  • 傻瓜安装即可

11 $IDEA$开发工具概述

  • $project$–>$module$–>$package$–>$class$
  • $package$包一般以倒序域名命名,如$com.ithema.$xxx
  • 创建$class$时不必写后缀,编译后的$class$文件位于$out$文件夹里

12 $IDEA$的配置,快捷键设置

  • 快捷键
    • $main,psvm,sout$用于快速键入代码
    • $Ctrl\ +\ D$ 复制当前行到下一行
    • $Ctrl\ +\ Y$ 删除当前行
    • $Ctrl\ +\ ALT\ +\ L$ 格式化代码
    • $ALT\ +\ SHIFT\ + \uparrow/\downarrow$ 上下移动当前代码
    • $Ctrl\ +\ /$ 注释当前行, $Ctrl\ +\ SHIFT\ +\ /$ 插入多行注释

13 $IDEA$的其他操作

  • 修改可能是$refactor$,再$rename$
  • 导入一般新建$module$到同一$project$后直接复制$codes$
  • 在$IDEA$里$remove$后检查是否真的删除

阶段总结:Java概述,快速入门,IDEA使用

14 基础语法-注释详解

  • 注释用于解释说明代码

  • 分类

    • 单行注释:$//$
    • 多行注释:$/\ /$
    • 文档注释:$/*\ /$

  • 注释特点:不影响程序运行,编译后的$class$文件中没有注释

  • 快捷键:

    • $Ctrl\ +\ /$ 当前行注释
    • $Ctrl\ +\ Shift\ + /$ 对选中代码多行注释

15 基础语法-字面量详解

  • 字面量是数据在程序中的书写格式,分为
    • 整数
    • 小数
    • 字符, 用单引号
    • 字符串, 用双引号
    • 布尔值, $true\ flase$
    • 空值, $null$
  • 常见特殊值
    • $true\ false\ null\ \textbackslash n \ \textbackslash t$

16 基础语法-变量详解

  • 用于储存数据, 本质是内存中的一块区域
  • 完整定义为: 数据类型 变量名称 = 数据;

17 基础语法-变量使用时的注意事项

  • 先声明后使用
  • 变量和数据的类型统一
  • 不可重名
  • 注意使用的变量需要初始化

18 基础语法-关键字,标识符

  • 关键字为$Java$内定的词, 不可用作类名, 变量名
  • 标识符即为名字, 不可以数字开头, 不可使用关键字

Java数组

02 静态初始化数组，数组的访问

  • 写法
    • 1.数据类型[] 数组名={元素1，元素2，…}；
    • 2.数据类型[] 数组名=new 数据类型[]{元素1，元素2，…}；
  • 数据类型[] 数组名 也可写成 数据类型 数组名[]
  • 数组属于引用数据类型，储存数组在内存中的地址
  • 访问
    • 元素：数组名[索引]
    • 长度：数组名.length
    • 最大索引：长度-1（长度>0）
    • 访问超出最大索引：索引越界异常

04 动态初始化数组，案例练习

  • 写法
    • 数据类型[] 数组名=new 数据类型[长度]；
  • 默认值
    • 整型 0
    • 浮点型 0.0
    • 布尔型 false
    • 引用型 null
  • 静态与动态
    • 静态适合元素量已知业务
    • 动态适合元素量未知业务

05 数组的内存执行原理

  • JVM分区
    • 方法区
    • 栈内存
    • 堆内存
  • int a与int[] arr=new int[3]的区别
    • a是变量，放在栈里，存储为a的值
    • new int[3]创建数组对象，在堆内存开辟3个int
    • arr是变量，在栈中，存储为堆内存的地址
  • 多个数组变量只向同一数组对象
    • 数组变量储存的是同一地址
    • 用任意数组变量修改数组内元素，其他均受影响
  • 存null表示没有指向对象，此时不能访问

Java方法

01 方法概述

  • 方法是一种语法结构,可以把一段代码封装生一个功能，以便重复调用

  • 完整格式

  • 执行：方法名称（实参）；

  • 好处

    • 1.提高代码复用性
    • 2.提高开发效率
    • 3.使程序逻辑更加清晰

02 方法定义的其他形式，常见问题

  • 不需要接受数据：取消形参，不输实参
  • 不需要返回数据：取消返回值，类型写void
  • 常见问题
    • 方法在类的位置无所谓，但不能在方法中定义方法、
    • 返回值写void时不能return 数据；除外都要return 对应类型数据
    • return 之后代码无效
    • 方法不调用不会执行，实参严格匹配形参列表
    • 有返回值的方法
      • 1.可以定义变量接受结果
      • 2.直接输出调用
      • 3.直接调用
    • 无返回值的方法
      • 只能直接调用

03 方法的案例讲解

  • 定义方法时关注的重点
    • 是否接受数据（决定形参列表）
    • 是否返回数据（决定返回值类型）

04 方法在计算机中的执行

  • 方法的运行区域在栈内存
  • 栈的特点：先进后出
  • 方法为什么要在栈中运行自己：保证方法调用完另一个方法时可以回来

02 分支结构：if，switch，switch穿透性

  • if可以根据条件选择执行某段程序
  • 可以写作
    • if
    • if else
    • if else
      • if else…
  • if泛用性强于swtich，switch在范围内性能较好
  • switch注意事项
    • 支持byte,short,int,char,枚举,String；不支持double,float,long
    • case只能是字面量不能是变量，不能重复
    • 正常使用要写break
  • 穿透性可用于多种入口的情形，简化代码

03 循环结构：for循环，for循环案例

  • for（初始化；循环条件；迭代语句）{循环内容}

04 循环结构：while循环，珠穆朗玛峰案例

  • while（条件）{循环；迭代}

05 循环结构：do-while循环
