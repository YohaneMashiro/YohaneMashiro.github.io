预备队训练week7

本周简述

复习了dfs和bfs，写了一点cf的题看了一些杂项（欧拉图，树知识）

CF1063B

解：

思路

明显的bfs，但是要考虑答案如何产生

如果简单地用vis数组判断，容易发生优解后遍历但是因为vis==1无法更新答案的情况

所以一个vis不足以否定劣解，发现优解和剩余移动次数是相关的

考虑一种一定劣解，即此前的剩余移动次数（左，右）均大于当前状态，舍去

事实上这样剪一下枝就可以跑过该题

code

反思：

bfs需要剪枝，但是不一定就是vis单一判据

剪枝时从最容易排除的状态开始，逐渐深入（或者就和这题一样直接跑过了）
