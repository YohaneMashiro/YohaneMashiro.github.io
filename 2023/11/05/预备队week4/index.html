tag：背包，存在性dp

思路

code

P8742

解：

思路

看出是dp，设dp[i] [j]表示前i种砝码可能表示的重量

边界dp[i] [0]=1;

转移，放在左边看作+，放在右边看作-，分三种：

  • 1.不放i，dp[i] [j]|=dp[i-1] [j]
  • 2.放在左边，注意此时有两种前驱情况，用绝对值表示 dp[i] [j]|=dp[i-1] [abs(j-w[i])]
  • 3.放在右边，dp[i] [j]|=dp[i-1] [j+w[i]]

code

反思：

思考数字上的本质问题，就能够发现是背包模型

P7404

解：

思路

容易发现本质是操作差分数组

为了获得题意序列，需要差分数组前一段均为正数，后一段均为负数

发现以上目的恰好符合差分数组的操作规律，即左端+1右端-1，那么就每次选min{前面的第一个负数变成正数，后面的倒数第一个正数变为负数}计入答案

用双指针实现即可

code

反思：

要抓住目标数组的本质特征，要加强实现能力，即操作双指针
