7-1 区间选点

tag：贪心，多角度

解：

思路

以右端点排序，相等时左端点越靠右越优先

从左至右判断区间：

  • 1.没有被包含已有点，将该区间右端点加入
  • 2.上一个右端点在区间内，下一个

7-2 添加括号

tag：构造，性质，贪心

解：

思路

发现a1总是分子，a2总是分母，其他均可在分子

观察判断条件为整数与否，转化为整除问题，明显分母越少越好

故仅用a2作分母，判断能否约分为1，即化为整数

code

反思：

构造题要分析本质，在本题中就是一个大的除法

要善于发现数的特征，如可以构造为a1*a3…an/a2

7-5 线性变换

tag：规律，模拟

解：

思路

发现几个性质

  • 1.由p的表达式，每一种p输出的结果一定
  • 2.p每次%n，那么p的种类最多n种
  • 3.x的每次增加只与p的值有关，与n等元素无关

综合，得到结论：p的取值必定循环，且大小不长于n；x的贡献在p的循环内一致

模拟找环处理即可

code

反思

细致观察一些小性质，抓住合起来的规律

写代码先捋清思路，涉及下标计算时要谨慎精确，使用变量做循环条件时要格外注意
