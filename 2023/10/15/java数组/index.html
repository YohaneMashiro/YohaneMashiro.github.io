Java数组

02 静态初始化数组，数组的访问

  • 写法
    • 1.数据类型[] 数组名={元素1，元素2，…}；
    • 2.数据类型[] 数组名=new 数据类型[]{元素1，元素2，…}；
  • 数据类型[] 数组名 也可写成 数据类型 数组名[]
  • 数组属于引用数据类型，储存数组在内存中的地址
  • 访问
    • 元素：数组名[索引]
    • 长度：数组名.length
    • 最大索引：长度-1（长度>0）
    • 访问超出最大索引：索引越界异常

    • 数据类型[] 数组名=new 数据类型[长度]；
  • 默认值
    • 整型 0
    • 浮点型 0.0
    • 布尔型 false
    • 引用型 null
  • 静态与动态
    • 静态适合元素量已知业务
    • 动态适合元素量未知业务

05 数组的内存执行原理

  • JVM分区
    • 方法区
    • 栈内存
    • 堆内存
  • int a与int[] arr=new int[3]的区别
    • a是变量，放在栈里，存储为a的值
    • new int[3]创建数组对象，在堆内存开辟3个int
    • arr是变量，在栈中，存储为堆内存的地址
  • 多个数组变量只向同一数组对象
    • 数组变量储存的是同一地址
    • 用任意数组变量修改数组内元素，其他均受影响
  • 存null表示没有指向对象，此时不能访问
