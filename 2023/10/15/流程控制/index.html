流程控制

02 分支结构：if，switch，switch穿透性

  • if可以根据条件选择执行某段程序
  • 可以写作
    • if
    • if else
    • if else
      • if else…
  • if泛用性强于swtich，switch在范围内性能较好
  • switch注意事项
    • 支持byte,short,int,char,枚举,String；不支持double,float,long
    • case只能是字面量不能是变量，不能重复
    • 正常使用要写break
  • 穿透性可用于多种入口的情形，简化代码

03 循环结构：for循环，for循环案例

  • for（初始化；循环条件；迭代语句）{循环内容}
  • 应用
    • 重复劳动
    • 灵活执行

04 循环结构：while循环，珠穆朗玛峰案例

  • while（条件）{循环；迭代}
  • while与for
    • 功能一致
    • 具体次数用for，不确定用while

05 循环结构：do-while循环

  • do-while是执行后判断，可以保证至少执行一次

06 死循环，循环嵌套

  • 死循环可用于服务器程序
  • 嵌套解决复杂问题

07 跳转关键字：break，continue

  • break结束当前循环，结束swtich分支
  • continue前往下次循环

08 随机数Random，Random案例

  • Random
    • 1.导包：import java.util.Random;
    • 2.Random r=new Random();
    • 3.int num=r.nextInt(10);
  • 生成65-91：r.nextInt(27)+65;
